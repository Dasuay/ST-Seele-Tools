<h3>注意事项</h3>

1. 模型选择：目前最好用的"AI扮演模型"仍然是"gemini-2.5-pro"，同时Dasuay大人配置的"预设"也是用于"gemini-2.5-pro"的，小伙伴们如想体验其他模型请自行导入相应的"预设"哦~
2. 手动保存：为了节省大家的流量，SeeleTavern预装了"手动保存"插件，"每10分钟"才会定时保存一次聊天数据。因此在"退出 / 刷新"前记得在左下角的 <span class="fa-solid fa-magic-wand-sparkles"></span> 选择"保存聊天"哦，不然下次游玩的时候，发现聊天记录不见了可就不好了~
3. 聊天文件：由于此账号是"开放"给大家游玩的，因此可能存在"你想玩的"角色卡已经有"其他玩家"在玩的情况。因此在游玩时如果看到角色卡"已经有聊天记录"的话，请点击左下角的 <span class="fa-solid fa-bars"></span> 选择"管理聊天文件"，然后"创建 / 选择"你的记录进行游玩吧，"不要动别人"的聊天记录哦~
4. 玩家设定：点击右上角的 <span class="fa-solid fa-face-smile"></span>  可以创建"你的"角色设定，增加游玩时的沉浸感~
5. 希儿工具箱：Dasuay大人为酒馆定制的工具插件，在左下角的 <span class="fa-solid fa-magic-wand-sparkles"></span> 选择"希儿工具箱"来打开，具体功能自行探索~
6. 记忆表格：预装的记忆插件，通过"总结上下文"内容来有效改善AI健忘的问题，点击右上角的 <span class="fa-solid fa-table"></span> 打开表格界面，详细教程可参考下方"参考资料"中的链接
7. 行动选择：在每次对话后让AI自己"生成选项"，如果不需要的话可以在左上角的  <span class="fa-solid fa-sliders"></span> 中下滑找到"✅行动选择栏"进行关闭
